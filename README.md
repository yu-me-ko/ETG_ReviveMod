# Revive Mod（复活系统增强）

一个mod使你可以在联机模式下救援你倒下的队友！

---

## ✨ 功能特点

* 使用 **交互键** 可复活倒地队友
* 引入 **递增惩罚复活机制**

  * 第一次复活：满血
  * 后续复活：逐渐减少血量
  * 保底至少保留一定血量
* 灵魂将在复活前**移动到尸体附近**

---

## 📦 依赖项

本 Mod 需要以下前置：

* BepInEx
* ModTheGungeonAPI

（r2modman 会自动安装依赖）

---

## 📥 安装方法（推荐）

### 使用 r2modman / Thunderstore Mod Manager

1. 安装 r2modman
2. 搜索本 Mod
3. 点击安装（依赖会自动安装）
4. 启动游戏（Modded）

---

## 🛠️ 手动安装

1. 安装 BepInEx
2. 将 DLL 放入：

```
BepInEx/plugins/
```

3. 启动游戏

## ❗ 注意事项

* 复活过程中移动可能会导致复活失败
* 扣血可能因短暂无敌帧存在轻微延迟


# Revive Mod

A gameplay mod that allow you to revive your dead friend in cooperation mode.

## ✨ Features

- Revive teammates with **interaction (E key)**
- Revive system with **scaling penalty**
  - First revive: Full health
  - Subsequent revives: Lose increasing health
  - Minimum health guaranteed
- Soul will **move to corpse before revive**
- Visual revive indicator (UI bar / feedback)

## 📦 Dependencies

- BepInEx
- ModTheGungeonAPI

## 📥 Installation (r2modman)

1. Install **r2modman / Thunderstore Mod Manager**
2. Install dependencies automatically
3. Install this mod
4. Launch the game via mod manager

## 🛠️ Manual Installation

1. Install BepInEx
2. Put DLL into:

```
BepInEx/plugins/
```

3. Run the game

## ❗ Notes

- Revive cancels if movement detected
- Health reduction may be delayed due to invincibility frames
